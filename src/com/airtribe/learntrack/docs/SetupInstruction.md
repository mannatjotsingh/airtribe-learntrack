# Setup Instructions

## JDK Version

* JDK 21 

## Steps to Run

1. Install Java JDK
2. Set JAVA_HOME environment variable
3. Open terminal in project folder

### Compile

javac -d . src/com/airtribe/learntrack/ui/Main.java

### Run

java com.airtribe.learntrack.ui.Main
