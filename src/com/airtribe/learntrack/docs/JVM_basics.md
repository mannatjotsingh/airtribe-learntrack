# JVM Basics

## What is JDK, JRE, JVM?

* JDK (Java Development Kit): Provides tools required to develop Java applications such as compiler (javac).
* JRE (Java Runtime Environment): Provides environment to run Java programs.
* JVM (Java Virtual Machine): Executes Java bytecode and converts it into machine-specific instructions.

## What is Bytecode?

Bytecode is the intermediate code generated after compiling Java source code using javac. It is platform-independent and executed by the JVM.

## Write Once, Run Anywhere

Java follows the principle of "Write Once, Run Anywhere". This means that Java code can run on any system that has a JVM installed, without modification.
